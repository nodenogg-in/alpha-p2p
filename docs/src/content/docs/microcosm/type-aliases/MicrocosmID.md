> **MicrocosmID**: `Opaque`\<`string`, `"MicrocosmID"`\>

A unique identifier for a Microcosm.
A microcosm ID contains a-z, 0-9, or dot, does not have consecutive dots, and does not end with a dot.
```
microcosm
example.microcosm
another.example.microcosm.1
```
